Welcome to the hashban.js demo. To see the demo in action, click one of the links above.

If you're using Google Chrome, you'll need to be viewing this on an actual webserver, and not via file:// due to Access-Control-Allow-Origin issues. To create a simple webserver with python, serving the demo at localhost:8000, type:

        > cd hashban.js/demo
        > python -m SimpleHTTPServer
